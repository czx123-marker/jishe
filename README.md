# VideoLingo：基于 AI 视频翻译的沉浸式语言学习平台

本项目是一个创新的语言学习平台，它将先进的 AI 视频翻译技术与个性化的学习功能相结合，为用户（特别是学习中文的外国人和学习外语的中国人）提供一种沉浸式、情景化的学习体验。

平台的核心思想是：**将任何视频转化为一本生动的、可交互的语言学习教材。**

## 核心功能

### I. AI 内容处理引擎

这部分是学习内容的来源，负责将原始视频转化为结构化的学习材料。

- **📺 网页用户界面**：提供简单易用的 Web 界面，方便用户上传视频和管理翻译任务。
- **🎤 人声/背景音分离**：利用 `demucs` 模型自动分离人声和背景音，为语音识别提供更纯净的音频，显著提高准确率。
- **✍️ 自动语音识别 (ASR)**：使用业界领先的 `faster-whisper` 模型将视频中的语音精准转换成带时间戳的文本。
- **🌍 AI 驱动的翻译**：调用大语言模型 (LLM) API 进行高质量的文本翻译，并支持用户提供自定义术语表 (`custom_terms.xlsx`) 以确保专业词汇的准确性。
- **🎬 双语字幕生成**：自动生成标准格式 (`.srt`) 的双语或单语字幕文件，方便用户对照学习。

### II. 语言学习系统

这部分是平台的核心，提供了围绕视频内容设计的学习和互动功能。

- **👤 用户认证系统**：提供完整的用户注册和登录功能，用于追踪和保存每个用户的学习进度、词汇本等个性化数据。
- **📚 核心词汇提取与管理**：系统能从翻译好的字幕中自动提取核心词汇，并呈现在用户的个人词汇本 (`vocabulary.html`) 中。
- **✍️ 互动练习与测试**：平台提供专门的练习页面 (`lianxiti.html`)，能根据用户学习过的词汇动态生成练习题，帮助用户巩固记忆。
- **🧭 清晰的学习导航**：通过导航页，用户可以方便地在视频学习、词汇复习和在线练习等不同功能模块间切换。

## 实现流程

整个平台的工作流分为两大模块：**内容处理**和**在线学习**，它们被整合在同一个应用中。

### 模块一：AI 内容处理流程

1.  **上传与初始化**：用户在前端上传视频，`app.py` (主应用) 接收文件并创建处理任务。
2.  **音频预处理**：调用 `demucs` 分离出纯净的人声音频。
3.  **语音转文本**：`core/_2_asr.py` 使用 `faster-whisper` 模型进行语音识别，生成带时间戳的原文。
4.  **智能分段**：`core/_3_*.py` 模块利用 NLP 技术对文本进行语义切分。
5.  **AI 翻译**：`core/_4_2_translate.py` 调用 LLM API 进行翻译，同时应用 `custom_terms.xlsx` 中的自定义术语。
6.  **字幕与词汇生成**：
    - `core/_6_gen_sub.py` 生成双语字幕文件。
    - 在此之后，系统会进一步分析文本，**提取核心词汇**，并将这些词汇与用户信息关联，存入后端的数据库中。

### 模块二：用户在线学习流程

1.  **用户登录**：用户通过 `frontend/login.html` 页面进行登录，由主应用 `app.py` 内集成的 `AuthManager` 进行验证。
2.  **学习导航**：登录后，用户可以通过导航页选择学习模块。
3.  **词汇学习**：
    - 用户访问 `frontend/vocabulary.html`。
    - 该页面通过 API 从主应用获取该用户的个人词汇列表，并在前端展示出来。
4.  **巩固练习**：
    - 用户进入 `frontend/lianxiti.html`。
    - 页面从主应用 API 获取为该用户量身定制的练习题。
    - 用户提交答案，后端进行判断并返回结果，完成学习闭环。

## 项目结构概览

- `/app.py`: **项目主入口**，一个 Flask Web 应用，整合并调度所有功能。
- `/backend/`: 作为**功能模块**，提供用户系统、数据库交互等能力。
    - `auth_system.py`: 处理用户认证的类。
    - `users.db`: SQLite 数据库文件，存储用户信息和学习数据。
- `/frontend/`: 包含所有**用户学习功能**的静态 HTML 页面（由 Flask 渲染）。
    - `login.html`: 登录页。
    - `vocabulary.html`: 个人词汇本页。
    - `lianxiti.html`: 在线练习页。
- `/core/`: 包含所有核心**内容处理**逻辑的模块（ASR、翻译、字幕生成等）。
- `/templates/`: 主应用 (`app.py`) 使用的 Flask 模板。
- `/uploads/` & `/output/`: 分别存放用户上传的视频和处理后的输出文件。

## 如何使用

本项目是一个单体 Web 应用，所有功能（包括后端用户系统）都通过一个命令启动，非常方便。

1.  **环境配置**：
    - 克隆本仓库到本地。
    - 安装所有 Python 依赖：`pip install -r requirements.txt`。
    - 检查并编辑 `config.yaml` 或 `config.json` 文件，确保其中已填入您的大语言模型 API Key 等必要信息。

2.  **启动服务**：
    - 打开一个终端，确保位于项目的根目录下。
    - 运行以下命令：
      ```bash
      python app.py
      ```
    
3.  **访问应用**：
    
    - 启动成功后，终端会显示应用运行的地址（默认为 `http://127.0.0.1:5001`）。
    - 打开浏览器访问该地址，即可开始使用平台的全部功能，包括上传视频、登录、查看词汇本和进行练习。
